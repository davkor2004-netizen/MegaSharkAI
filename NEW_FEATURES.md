# 🆕 Новые функции MegaSharkAI

## 📊 Экспорт в Excel

**Как использовать:**
1. Откройте **📦 Товары**
2. Примените фильтры (маркетплейс, тип товара)
3. Нажмите кнопку **"📥 Экспорт в Excel"**
4. Файл скачается автоматически

**Что в файле:**
- ID, Название, Бренд, Категория
- Цена, Старая цена, Скидка %
- Рейтинг, Отзывы, Маркетплейс
- Тип (свои/конкуренты)
- Дата добавления

**Формат:** `megashark_products_YYYYMMDD_HHMMSS.xlsx`

---

## 📈 История цен

**Как посмотреть:**
1. Откройте **📦 Товары**
2. Найдите нужный товар
3. Нажмите кнопку **📈** в строке товара
4. Откроется модальное окно с историей

**Что показывает:**
- Дата и время изменения
- Новая цена
- Старая цена (зачёркнута)
- Скидка в процентах

**Когда обновляется:**
- При каждом парсинге товара
- Если цена изменилась — создаётся новая запись
- Хранится до 100 последних записей

**Пример:**
```
Дата          Цена    Старая   Скидка
15.04 14:30   1999₽   2999₽    -33%
14.04 10:15   2499₽   2999₽    -17%
13.04 09:00   2999₽   —        —
```

## 💾 Кэширование парсинга

**Как работает:**
1. При парсинге URL сохраняется в Redis
2. При повторном запросе того же URL — данные берутся из кэша
3. Кэш хранится 24 часа

**Преимущества:**
- ⚡ Мгновенный результат (вместо 3-5 секунд)
- 🛡️ Защита от блокировок WB
- 💰 Экономия ресурсов

**Как проверить:**
1. Спарсите товар: `https://www.wildberries.ru/catalog/24319630/detail.aspx`
2. Спарсите его ещё раз через 1 минуту
3. Второй раз ответ будет мгновенным с пометкой `(cached)`

**Очистка кэша:**
```python
from app.services.cache_service import cache_service
await cache_service.clear_parse_cache()
```
