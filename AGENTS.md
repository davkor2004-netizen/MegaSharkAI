# Инструкции для AI-агента: Проект MegaSharkAI

## 1. Контекст проекта
Этот проект — AI-ассистент для продавцов маркетплейсов MegaSharkAI. Он анализирует товары, конкурентов и помогает с ценообразованием.

## 2. Технологический стек
- **Бэкенд**: Python 3.12 + FastAPI + Celery + Redis
- **Фронтенд**: SvelteKit + shadcn-svelte
- **База данных**: PostgreSQL + pgvector
- **Парсинг**: Playwright (Python)
- **Инфраструктура**: Docker + Docker Compose

## 3. Структура проекта (Монорепозиторий)
- `backend/` — Весь код FastAPI, Celery и логика бэкенда.
- `frontend-app/` — Приложение личного кабинета на SvelteKit.
- `widget/` — Код встраиваемого на чужие сайты виджета.

## 4. Правила написания кода (Code Rules)
- **Пиши код с подробными комментариями на русском языке**.
- Для Python: используй 4 пробела для отступов.
- Для Svelte: используй 2 пробела для отступов.
- Имена переменных и функций должны быть понятными (`get_product_price`, а не `gpp`).

## 5. Основные команды для запуска
- Запуск всей инфраструктуры: `docker compose up -d`
- Запуск бэкенда в режиме разработки: `cd backend && uvicorn app.main:app --reload`
- Запуск фронтенда в режиме разработки: `cd frontend-app && npm run dev`

## 6. Важные запреты (Restrictions)
- **НИКОГДА** не публикуй в коде секретные ключи (API-ключи, пароли). Используй переменные окружения (файл `.env`).
- **НЕ УСТАНАВЛИВАЙ** новые библиотеки без моего явного разрешения.
- **НЕ УДАЛЯЙ** файлы без моего подтверждения.
